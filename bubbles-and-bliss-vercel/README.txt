Bubbles & Bliss website package

Files included:
- bubbles-and-bliss.html  -> main website file
- vercel.json             -> Vercel routing config so the site opens cleanly

How to deploy on Vercel:
1. Upload this folder to a GitHub repository.
2. Import the repository into Vercel.
3. Vercel will publish the site automatically.
4. Because of vercel.json, visiting the root URL will load bubbles-and-bliss.html.

Before going live, update these items inside bubbles-and-bliss.html:
- Replace temporary placeholder product images with actual downloaded Instagram product photos.
- Update WhatsApp number if needed.
- Add real prices where applicable.
- Optionally connect the inquiry form to a form backend.
